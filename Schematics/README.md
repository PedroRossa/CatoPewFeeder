* 1x Node MCU 8266
* 2x Servo Motor
* 4x Cathod Leds
* 4x 330 Ω resistor
* 2x Push Buttons
* 2x 10k Ω resistors
* 1x Ultrassonic sensor (HC-SR04)
* 1x 9v battery connector
* 1x 9v battery
* 1x LM7805
